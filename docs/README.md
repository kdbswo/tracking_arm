## 환경 설정(.env)

루트 디렉터리의 `.env` 파일 하나만 수정하면 로봇/스트림 주소 전체가 바뀐다. 기본 내용은 다음과 같다.

```ini
JETCOBOT_STREAM_URL=http://192.168.0.165:5000/stream
JETCOBOT_UDP_HOST=192.168.0.17
JETCOBOT_UDP_PORT=6240
```

ROS 노드/추적 앱은 시작 시 `.env`를 자동으로 읽어 환경 변수에 로드한다. 다른 주소를 사용하려면 `.env`를 편집하거나, 일회성으로는 기존처럼 셸에서 환경 변수를 export 하면 된다.
